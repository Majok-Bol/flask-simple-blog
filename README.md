 Flask Simple Blog

A beginner-friendly Flask + PostgreSQL blog application where you can create, read, update, and delete posts.  
This project is part of a learning roadmap for mastering full-stack development using Flask, Tailwind CSS, and PostgreSQL.

Core Features
- Create blog posts  
- View all posts on the homepage  
- View a single post  
- Edit posts  
- Delete posts  
- Store posts in PostgreSQL
